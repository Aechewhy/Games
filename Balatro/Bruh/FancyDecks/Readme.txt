Fancy Decks

-Steamodded is required to run this mod.

-It has been tested to work when it is the only mod loaded in game.

-If card deck textures do not appear, there is most likely a conflicting mod

-Any mod that uses the "Enhancers.png" file will need to be replaced by the "FancyDecks.png" image from this mod
   1. Copy, paste, rename "FancyDecks.png" to "Enhancers.png"
   2. Use new image to replace "Enhancers.png" in the conflicting mod folder
